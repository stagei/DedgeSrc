Dashboard Service Configuration

This is the appsettings.json file for the Dashboard service itself. Key sections include:

  • Dashboard.AlertPolling - Configure alert polling (enabled, interval, severity filter)
  • Dashboard - Server paths, ports, polling intervals
  • Authentication - User access control (FullAccess, Standard, Blocked)
  • Kestrel - Dashboard web server port configuration

Note: Changes to some settings (like Kestrel port) require a service restart.
