### 📝 Description

> Convert SQL DDL to Mermaid ERD diagrams automatically. Zero-configuration tool that transforms database schemas (CREATE TABLE statements) into beautiful Mermaid entity-relationship diagrams. Supports SQL Server (T-SQL), PostgreSQL, MySQL, SQLite, Oracle, and ANSI SQL. Perfect for documentation, visualization, and database design. Includes embedded Python runtime - no external dependencies required!
