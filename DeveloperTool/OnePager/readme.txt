A flexible, modern one-page business theme with fully configurable content via the WordPress Customizer.

== Description ==

OnePager is a professional one-page WordPress theme designed for businesses, agencies, freelancers, and service providers. Every text element, icon, and section is editable through the WordPress Customizer — no coding required.

**Key Features:**

* **13 Built-in Sections** — Hero, About, Expertise, Services, Process, Team, Partners, Enterprise/B2B, Pricing, Contact, FAQ, Call-to-Action, and Thank You
* **Fully Configurable Content** — Every heading, paragraph, button, badge, and icon is editable via the Customizer with live preview
* **150+ Font Awesome Icons** — All icon fields use dropdown selectors with categorized, labeled icons
* **Conditional Rendering** — Leave any text field empty and the entire element (including its icon) disappears; the layout automatically reorganizes
* **Auto-Reflowing Grids** — CSS Grid with `auto-fit` ensures layouts adapt when elements are hidden
* **Custom Post Types** — Services, Team Members, and FAQ items are managed through dedicated admin menus
* **Contact Form 7 Ready** — Built-in CF7 integration with styled form output and a static HTML fallback
* **Theme Activation Setup** — On first activation, the theme creates a front page, navigation menu, and sample content automatically
* **Mobile Responsive** — Fully responsive design with mobile menu, touch-friendly interactions, and optimized typography
* **Translation Ready** — All strings use WordPress i18n functions for easy localization
* **No Page Builder Required** — Everything works out of the box with the native WordPress Customizer
* **Custom Nav Walker** — CTA-styled last menu item for prominent call-to-action in the navigation

**Sections Overview:**

1. **Hero** — Full-viewport hero with animated title, description, two CTA buttons, and up to 3 customizable badges with icons
2. **About** — Two-column layout with image, lead text, paragraphs, and up to 3 statistics
3. **Expertise** — Section with intro text, 3 expertise cards with icons, a "Who is this for?" subsection, and a process overview
4. **Services** — Grid of service cards powered by a Custom Post Type with icon dropdowns
5. **Process** — 4-step numbered process on a dark background
6. **Team** — Staff/team profiles with photo, bio, credentials, and specialties from a Custom Post Type
7. **Partners** — 3-step partner process, partner logo grid, and a note section
8. **Enterprise** — B2B section with benefit badges, description, 3 service cards, statistics box, and CTA
9. **Pricing** — 3 pricing cards with a "Most Popular" badge, notes, and cancellation policy
10. **Contact** — Contact info (address, phone, email, hours) alongside a Contact Form 7 form or HTML fallback
11. **FAQ** — Accordion-style FAQ from a Custom Post Type
12. **Call to Action** — Gradient CTA section with phone/email buttons and a trust badge
13. **Thank You** — Hidden confirmation section that shows after form submission

== Installation ==

1. In your WordPress admin, go to **Appearance → Themes → Add New → Upload Theme**
2. Upload the `onepager.zip` file
3. Click **Install Now**, then **Activate**
4. The theme automatically creates a front page, navigation menu, and sample content
5. Go to **Appearance → Customize → OnePager Content** to edit all text, icons, and settings
6. Edit Team Members, Services, and FAQ items under their respective admin menus

== Frequently Asked Questions ==

= Do I need any plugins? =

The theme works without any plugins. For the contact form, we recommend installing **Contact Form 7** (free) — the theme will automatically create and style a form. Without CF7, a static HTML fallback form is displayed.

= Can I hide sections I don't need? =

Yes! Simply clear the text content for any element or section in the Customizer. Empty elements are automatically hidden, and the page layout reorganizes gracefully.

= Can I change the icons? =

Every icon in the theme has a dropdown selector in the Customizer with 150+ Font Awesome icons to choose from. You can also select "No icon" to remove an icon entirely.

= Is the theme translation-ready? =

Yes. All strings use WordPress internationalization functions (`__()`, `esc_html__()`, etc.) with the `onepager` text domain.

= Can I use this for a multi-page site? =

OnePager is designed as a one-page theme, but you can create additional pages in WordPress and link to them from the navigation menu.
