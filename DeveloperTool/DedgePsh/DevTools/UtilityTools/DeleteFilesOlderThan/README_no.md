# DeleteFilesOlderThan

**Kategori:** UtilityTools | **Tidsbesparelse:** ~16 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Slett filer eldre enn spesifiserte dager  
**Formål:** Automatisert filopprydding  
**Verdi:** Diskplasshåndtering

### Funksjonalitet
```powershell
.\DeleteFilesOlderThan.ps1 -Path "C:\Logs" -DaysOld 90 -Recurse
```

**Funksjoner:**
- Rekursivt søk
- Størrelsesterskel-alternativ
- Dry-run-modus
- Detaljert logging

### Tidsbesparelse
- **Lagring:** Frigjør GB med plass
- **Automatisering:** Planlagt opprydding
- **Forhindrer:** Disk full-problemer
- **Tidsbesparelse per kjøring:** ~16 minutter (manuelt arbeid eliminert)
