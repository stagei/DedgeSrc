## 🎯 Forretningsverdi

**Verdi:** Gjenbruk oppgavedefinisjoner på tvers av servere

- Oppdater stier
- Endre brukerkontoer
- Juster tidsplaner

**Bruksområde:** DEV → TEST → PROD oppgave-migrering
