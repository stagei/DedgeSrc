**Kategori:** AdminTools | **Tidsbesparelse:** ~39 min/kjøring

## 🎯 Forretningsverdi

**Funksjon:** Distribuer standard planlagte oppgaver til servere  
**Formål:** Automatisert server oppgave-provisjonering  
**Verdi:** Konsistent automatisering på tvers av alle servere

### Funksjonalitet
Distribuerer oppgaver:
- Backup-oppgaver
- Overvåkingsoppgaver
- Vedlikeholdsoppgaver
- Log-oppryddingsoppgaver
- Helsesjekkoppgaver

**Fra:** Sentralt oppgave-repository (XML-definisjoner)

### Tidsbesparelse
- **Konsistens:** Alle servere har påkrevde oppgaver
- **Tid:** 2 timer manuelt → 10 min automatisert
- **Nye servere:** ~30/år
- **Tidsbesparelse per kjøring:** ~39 minutter (manuelt arbeid eliminert)
