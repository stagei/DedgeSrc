# Chg-Pass

**Category:** AdminTools  
**Complexity:** 🟢 Low  
**Time Saved:** ~6 min/run

---

## 🎯 Business Value

**Function:** Simple Windows password change tool  
**Method:** Wrapper for `net user` command  
**Value:** User-friendly password changes

### Functionality
Quick password change without complex AD tools

**Usage:**
```powershell
.\Chg-Pass.ps1
# Prompts for: username, old password, new password
```

### Time Saved
- **User-friendly:** Simple interface
- **Reduces support:** Self-service password changes
- **Time Saved per execution:** ~6 minutes (manual work eliminated)
