# Chg-Pass

**Kategori:** AdminTools  
**Kompleksitet:** 🟢 Lav  
**Tidsbesparelse:** ~6 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Enkelt Windows passordendring-verktøy  
**Metode:** Wrapper for `net user` kommando  
**Verdi:** Brukervennlig passordendring

### Funksjonalitet
Rask passordendring uten komplekse AD-verktøy

**Bruk:**
```powershell
.\Chg-Pass.ps1
# Spør etter: brukernavn, gammelt passord, nytt passord
```

### Tidsbesparelse
- **Brukervennlig:** Enkelt grensesnitt
- **Reduserer support:** Selvbetjening passordendringer
- **Tidsbesparelse per kjøring:** ~6 minutter (manuelt arbeid eliminert)
