# WinUninstall

**Kategori:** AdminTools | **Tidsbesparelse:** ~167 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Avinstaller Windows-applikasjoner  
**Formål:** Programvare-fjerningsautomatisering  
**Verdi:** Rene avinstallasjoner

### Funksjonalitet
- Lister installerte applikasjoner
- Stille avinstallasjon (MSI/EXE)
- Fjerner registeroppføringer
- Rydder opp etterlatte filer
- Logger alle handlinger

**Støtter:** WinGet, Chocolatey, native installatører

### Tidsbesparelse
- **Automatisering:** Skriptet programvare-fjerning
- **Renslighet:** Komplett avinstallasjon
- **Tidsbesparelse per kjøring:** ~167 minutter (manuelt arbeid eliminert)
