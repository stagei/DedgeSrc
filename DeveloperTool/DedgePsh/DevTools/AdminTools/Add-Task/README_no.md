# Add-Task

**Kategori:** AdminTools | **Tidsbesparelse:** ~46 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Forenklet wrapper for opprettelse av planlagte oppgaver  
**Verdi:** Raskere oppgaveopprettelse enn GUI

### Funksjonalitet
Kommandolinjegrensesnitt for å opprette planlagte oppgaver med vanlige mønstre

**Funksjoner:**
- Forhåndsdefinerte tidsplanmønstre (Daglig, Ukentlig, Månedlig)
- Vanlige tidspunkter
- Bruker/System kontekstvalg
- Feilhåndtering

**Bruk raskere enn:**
- Oppgaveplanlegger GUI (5-10 min)
- Manuell XML-opprettelse
- PowerShell Register-ScheduledTask med alle parametere

### Tidsbesparelse
- **Tid spart:** 5 min → 30 sek per oppgave
- **Admin-produktivitet:** ~50 oppgaver/måned
- **Tidsbesparelse per kjøring:** ~46 minutter (manuelt arbeid eliminert)
