# Inst-Psh

**Kategori:** AdminTools | **Tidsbesparelse:** ~40 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Installer PowerShell-moduler og avhengigheter  
**Formål:** Automatisert PowerShell-miljøoppsett  
**Verdi:** Konsistent PowerShell-konfigurasjon

### Funksjonalitet
Installerer:
- Tilpassede FK PowerShell-moduler
- Påkrevde eksterne moduler (PSWindowsUpdate, etc.)
- Modulavhengigheter
- Setter PSModulePath

**Distribusjon:** Kan distribuere til flere servere

### Tidsbesparelse
- **Onboarding:** Øyeblikkelig PowerShell-miljø
- **Konsistens:** Alle servere identisk oppsett
- **Tid:** 30 min → 5 min
- **Tidsbesparelse per kjøring:** ~40 minutter (manuelt arbeid eliminert)
