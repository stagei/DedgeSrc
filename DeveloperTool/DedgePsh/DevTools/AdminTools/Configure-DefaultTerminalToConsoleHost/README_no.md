## 🎯 Forretningsverdi

**Funksjon:** Sett Windows Terminal standard til Console Host  
**Formål:** Brukerpreferanse-konfigurasjon  
**Verdi:** Konsistent terminal-opplevelse

### Funksjonalitet
Konfigurerer Windows Terminal til å bruke Console Host som standard

**Hvorfor:** Noen brukere foretrekker klassisk konsoll over Windows Terminal
