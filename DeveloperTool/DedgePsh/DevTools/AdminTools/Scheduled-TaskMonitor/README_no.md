# Scheduled-TaskMonitor

**Kategori:** AdminTools | **Tidsbesparelse:** ~126 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Overvåk planlagte oppgaver helse og feil  
**Formål:** Proaktiv oppgavefeil-deteksjon  
**Verdi:** Forhindrer tapte automatiserte jobber

### Funksjonalitet
Overvåker:
- Oppgave-kjøringsstatus
- Mislykkede oppgaver
- Deaktiverte oppgaver
- Oppgaver ikke kjørt på plan
- Langvarige oppgaver

**Varsler:** E-post ved feil

### Tidsbesparelse
- **Pålitelighet:** Fang mislykket automatisering tidlig
- **Nedetidsforebygging:** ~15 hendelser/år forhindret
- **Tid:** Manuell sjekking → automatisert overvåking
- **Tidsbesparelse per kjøring:** ~126 minutter (manuelt arbeid eliminert)
