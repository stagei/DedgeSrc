# Inst-WinApp

**Kategori:** AdminTools | **Kompleksitet:** 🟢 Lav | **Tidsbesparelse:** ~10 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Installer Windows-applikasjoner  
**Metode:** Wrapper for SoftwareUtils installasjon  
**Verdi:** Forenklet app-installasjon

### Bruk
```powershell
# Installer spesifikk app
.\Inst-WinApp.ps1 -appName "AppName"

# Oppdater FkPsh apps
.\Inst-WinApp.ps1  # Ingen parametere
```

### Tidsbesparelse
- **Brukervennlig:** Enkelt grensesnitt vs komplekse winget-kommandoer
- **Standardisering:** Konsistent installasjonsmetode
- **Tidsbesparelse per kjøring:** ~10 minutter (manuelt arbeid eliminert)
