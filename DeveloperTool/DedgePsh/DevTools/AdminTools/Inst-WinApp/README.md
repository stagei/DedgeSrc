# Inst-WinApp

**Category:** AdminTools | **Complexity:** 🟢 Low | **Time Saved:** ~10 min/run

---

## 🎯 Business Value

**Function:** Install Windows applications  
**Method:** Wrapper for SoftwareUtos Installation  
**Value:** Simplified app Installation

### Usage
```powershell
# Install specific app
.\Inst-WinApp.ps1 -appName "AppName"

# Update FkPsh apps
.\Inst-WinApp.ps1  # No parameters
```

### Time Saved
- **User-friendly:** Simple interface vs complex winget commands
- **Standardization:** Consistent Installation method
- **Time Saved per execution:** ~10 minutes (manual work eliminated)
