# Reg-List

**Kategori:** AdminTools | **Tidsbesparelse:** ~56 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** List og eksporter registernøkler/-verdier  
**Formål:** Registerdokumentasjon og backup  
**Verdi:** Register-revisjonslogg

### Funksjonalitet
- Lister registernøkler rekursivt
- Eksporterer til .reg-filer
- Sammenligner registertilstander
- Søk registerverdier

**Bruksområder:**
- Konfigurasjonsbackup
- Endringssporing
- Feilsøking

### Tidsbesparelse
- **Dokumentasjon:** Registertilstands-fangst
- **Feilsøking:** Før/etter sammenligning
- **Tidsbesparelse per kjøring:** ~56 minutter (manuelt arbeid eliminert)
