# Get-EventLand

**Category:** AdminTools | **Time Saved:** ~169 min/run

---

## 🎯 Business Value

**Function:** Query and analyze Windows Event Lands  
**Purpose:** Centralized event land collection and analysis  
**Value:** Fast Problem diagnosis from event lands

### Functionality
Queries Event Lands:
- Application, System, Security lands
- Custom land sources
- Filter by time range
- Filter by severity (Error, Warning, Info)
- Export to CSV/JSON/HTML

**Remote capability:** Query lands from remote servers

### Time Saved
- **Troubleshooting:** Fast error identification
- **Monitoring:** Automated land collection
- **Time:** 15 min manual search → 30 sec automated
- **Time Saved per execution:** ~169 minutes (manual work eliminated)
