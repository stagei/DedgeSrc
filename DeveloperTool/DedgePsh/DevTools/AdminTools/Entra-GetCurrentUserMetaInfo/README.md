# Entra-GetCurrentUserMetaInfo

**Category:** AdminTools | **Time Saved:** ~6 min/run

---

## 🎯 Business Value

**Function:** Get current user's Azure Entra ID (Azure AD) metadata  
**Value:** User information for automation

### Functionality
Queries Azure Entra ID and returns:
- DisplayName
- Email address
- Department
- Manager
- Job title
- Group memberships
- Office location

**Usage in automation:**
- Email notifications
- Manager approvals
- Department-specific landic
- User provisioning

### Time Saved
- **Automation:** User context in scripts
- **Accuracy:** Always current data from AD
- **Time Saved per execution:** ~6 minutes (manual work eliminated)
