# Entra-GetCurrentUserMetaInfo

**Kategori:** AdminTools | **Tidsbesparelse:** ~6 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Hent nåværende brukers Azure Entra ID (Azure AD) metadata  
**Verdi:** Brukerinformasjon for automatisering

### Funksjonalitet
Spør Azure Entra ID og returnerer:
- Visningsnavn
- E-postadresse
- Avdeling
- Leder
- Jobbtittel
- Gruppemedlemskap
- Kontorplassering

**Bruk i automatisering:**
- E-postvarsler
- Leder-godkjenninger
- Avdelingsspesifikk logikk
- Bruker-provisjonering

### Tidsbesparelse
- **Automatisering:** Brukerkontekst i scripts
- **Nøyaktighet:** Alltid oppdaterte data fra AD
- **Tidsbesparelse per kjøring:** ~6 minutter (manuelt arbeid eliminert)
