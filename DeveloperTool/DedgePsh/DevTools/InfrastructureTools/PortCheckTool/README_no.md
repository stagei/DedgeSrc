# PortCheckTool

**Kategori:** InfrastructureTools | **Tidsbesparelse:** ~150 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Test nettverksport-tilkobling  
**Formål:** Nettverksfeilsøking  
**Verdi:** Rask tilkoblingsdiagnose

### Funksjonalitet
Tester:
- TCP-porter
- UDP-porter
- Flere verter
- Vanlige tjenester (HTTP, HTTPS, SQL, RDP)
- Port-skanning

**Output:** Tilkoblingsmatrise

### Tidsbesparelse
- **Feilsøking:** Rask port-testing
- **Førdistribusjon:** Verifiser tilkobling
- **Brannmur-validering:** Port åpen/lukket-verifisering
- **Tidsbesparelse per kjøring:** ~150 minutter (manuelt arbeid eliminert)
