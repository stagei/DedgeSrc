Oppsett av maskin med FK powershell system for andre FK brukere

Følg disse trinnene for å sette opp maskinen din med nødvendig programvare og verktøy for å bruke FK powershell system.

1. Installer Visual Studio Code

Last ned og installer VS Code fra Microsoft Store:

Installer VS Code fra Microsoft Store

2. Åpne Visual Studio Code

Start VS Code fra startmenyen eller søk etter "Visual Studio Code" i Windows-søkefeltet.

3. Åpne en ny terminal i VS Code

Klikk på "Terminal" i toppmenyen og velg "New Terminal", eller bruk hurtigtasten Ctrl+` (control + backtick).

4. Kjør installasjonsskriptet

Kopier og lim inn følgende sti i terminalen:

C:\opt\src\DedgeSrc\DedgeSystemTools\Folders\DedgeCommon\Software\DedgePshApps\Init-Machine\Init-MachineOther.bat

Trykk Enter for å kjøre skriptet.

5. Følg instruksjonene på skjermen

Skriptet vil guide deg gjennom resten av installasjonen. Følg instruksjonene som vises i terminalen.

Merk: Hvis du støter på problemer under installasjonen, kontakt Geir Helge Starholm for hjelp.
