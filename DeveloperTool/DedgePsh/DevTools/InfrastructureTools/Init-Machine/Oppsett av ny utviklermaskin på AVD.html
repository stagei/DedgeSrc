Oppsett av ny utviklermaskin på AVD (VDI)

Følg disse trinnene for å sette opp din nye utviklermaskin med nødvendig programvare og verktøy.

🌐 1. Konfigurer DNS-suffix (kun ved nettverksproblemer)

Hvis du ikke kan nå dedge-server via Windows Kjør (Win+R), må DNS-suffixet settes manuelt. Åpne PowerShell som administrator og kjør:

Set-DnsClient -InterfaceAlias "Ethernet 7" -ConnectionSpecificSuffix "DEDGE.fk.no"

Tips: Verifiser tilkobling etterpå ved å åpne Win+R og skrive dedge-server. Hvis du ser en filutforsker med mapper, er nettverket klart. Interfacenavn kan variere — kjør Get-NetAdapter for å finne riktig navn.

💻 2. Åpne en ny CMD.EXE i Administrator-modus terminal fra startmenyen

⚙️ 3. Kjør installasjonsskriptet

Kopier og lim inn følgende sti i terminalen:

C:\opt\src\DedgeSrc\DedgeSystemTools\Folders\DedgeCommon\Software\DedgePshApps\Init-Machine\Init-MachineFkm.bat

Trykk Enter for å kjøre skriptet.

📦 4. Installer Microsoft Office

Åpne Edge-nettleseren og gå til Microsoft Office-portalen for å laste ned og installere Office:

Åpne Office-installasjon i Edge

Logg inn med din Dedge-konto og klikk Installer Office.

✅ 5. Følg instruksjonene på skjermen

Skriptet vil guide deg gjennom resten av installasjonen. Følg instruksjonene som vises i terminalen.

Merk: Hvis du støter på problemer under installasjonen, kontakt Geir Helge Starholm for hjelp.
