# Init-Machine

**Kategori:** InfrastructureTools | **Tidsbesparelse:** ~279 min/kjøring

---

## 🎯 Forretningsverdi

**Problem:** Nytt maskinoppsett = 4-6 timer manuelt arbeid  
**Løsning:** Automatisert komplett maskininitialisering  
**Verdi:** Fra bare OS til produksjonsklar på minutter

### Funksjonalitet
Komplett oppsett:
1. Windows-konfigurasjon
2. Programvareinstallasjon
3. Nettverkskonfigurasjon
4. Sikkerhetsoppsett
5. Brukerkontoer
6. Planlagte oppgaver
7. Overvåkingsagenter
8. Applikasjons-distribusjon

**Idempotent:** Kan kjøres på nytt trygt

### Tidsbesparelse
- **Tid:** 4-6 timer → 30 min
- **Kvalitet:** Null utelatte trinn
- **Nye maskiner:** ~60/år
- **Tidsbesparelse per kjøring:** ~279 minutter (manuelt arbeid eliminert)
