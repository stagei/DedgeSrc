Oppsett av ny server

Følg disse trinnene for å sette opp din nye server med nødvendig programvare og verktøy.

1. Koble til serveren med RDP

Bruk Remote Desktop (RDP) for å koble til serveren.

2. Åpne kommandolinjen som administrator

Høyreklikk på Start-menyen og velg "Kommandolinje (admin)" eller søk etter "cmd" i Windows-søkefeltet, høyreklikk og velg "Kjør som administrator".

3. Kjør installasjonsskriptet

Kopier og lim inn følgende sti i kommandolinjen:

C:\opt\src\DedgeSrc\DedgeSystemTools\Folders\DedgeCommon\Software\DedgePshApps\Init-Machine\Init-MachineFkm.bat

Trykk Enter for å kjøre skriptet.

4. Følg instruksjonene på skjermen

Skriptet vil guide deg gjennom resten av installasjonen. Følg instruksjonene som vises i kommandolinjen.

Merk: Hvis du støter på problemer under installasjonen, kontakt Geir Helge Starholm for hjelp.
