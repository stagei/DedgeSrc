# Generate-ServerShortcuts

**Kategori:** InfrastructureTools | **Tidsbesparelse:** ~7 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Generer RDP-snarveier for alle servere  
**Formål:** Rask servertilgang  
**Verdi:** Brukerproduktivitet

### Funksjonalitet
- Spør serverliste
- Oppretter RDP-snarveier
- Organiserer etter kategori
- Setter tilpassede ikoner
- Konfigurerer tilkoblingsinnstillinger

**Output:** Skrivebord/Startmeny-snarveier

### Tidsbesparelse
- **Produktivitet:** Ett-klikk server-tilgang
- **Admin-effektivitet:** Raskere operasjoner
- **Tidsbesparelse per kjøring:** ~7 minutter (manuelt arbeid eliminert)
