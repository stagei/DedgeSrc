## 🎯 Forretningsverdi

**Funksjon:** Opprett og konfigurer C:\opt-mappe-deling  
**Formål:** Standard distribusjonsdeling  
**Verdi:** Sentralisert distribusjonslokasjon

### Funksjonalitet
- Oppretter C:\opt-mappe
- Deler som `\\server\opt$`
- Setter tilganger
- Konfigurerer delingsinnstillinger
- Tester tilgang

**Standard:** Alle servere har opt$-deling
