# Db2-Commands

**Kategori:** DatabaseTools | **Tidsbesparelse:** ~62 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Vanlige Db2-kommandoer wrapper-bibliotek  
**Formål:** Forenklet Db2-operasjoner  
**Verdi:** Brukervennlig Db2-kommandogrensesnitt

### Funksjonalitet
Wrappere for:
- `db2 connect`
- `db2 list database directory`
- `db2 get db cfg`
- `db2 update db cfg`
- `db2 force applications all`
- `db2 backup database`
- `db2 restore database`

**PowerShell-vennlig:** Returnerer objekter, ikke tekst
