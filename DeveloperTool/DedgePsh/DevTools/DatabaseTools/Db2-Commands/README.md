# Db2-Commands

**Category:** DatabaseTools | **Time Saved:** ~62 min/run

---

## 🎯 Business Value

**Function:** Common Db2 commands wrapper library  
**Purpose:** Simplified Db2 operations  
**Value:** Easy-to-use Db2 command interface

### Functionality
Wrappers for:
- `db2 connect`
- `db2 list Database directory`
- `db2 get db cfg`
- `db2 update db cfg`
- `db2 force applications all`
- `db2 Backup Database`
- `db2 Restore Database`

**PowerShell-friendly:** Returns objects, not text

### Time Saved
- **Productivity:** No need to remember Db2 syntax
- **Automation:** Easier to script
- **Time Saved per execution:** ~62 minutes (manual work eliminated)
