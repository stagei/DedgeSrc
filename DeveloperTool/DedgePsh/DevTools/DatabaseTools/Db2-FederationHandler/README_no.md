## 🎯 Forretningsverdi

**Problem:** Manuell føderert database-oppsett = komplekst, feilutsatt

Konfigurerer føderasjon:
- Oppretter wrappere
- Definerer servere
- Mapper brukere
- Oppretter nicknames
- Tester tilkoblinger
- Gir tilganger

**Muliggjør:** Spørre mot flere databaser som én
