# Db2-AddCat

**Kategori:** DatabaseTools | **Tidsbesparelse:** ~76 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Katalogiser Db2-database  
**Formål:** Klientsideregistrering av database  
**Verdi:** Rask databasetilkoblingsoppsett

### Funksjonalitet
Katalogiserer:
- Node (server)
- Database
- Konfigurerer tilkoblingsparametere
- Tester tilkobling

**Kommando:** `db2 catalog tcpip node ... database ...`

### Tidsbesparelse
- **Hastighet:** Manuell katalogisering → automatisert
- **Nøyaktighet:** Korrekte parametere alltid
- **Tidsbesparelse per kjøring:** ~76 minutter (manuelt arbeid eliminert)
