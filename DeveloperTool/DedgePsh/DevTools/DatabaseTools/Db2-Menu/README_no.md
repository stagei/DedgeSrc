# Db2-Menu

**Kategori:** DatabaseTools | **Tidsbesparelse:** ~180 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Interaktiv meny for vanlige Db2-adminoppgaver  
**Formål:** Brukervennlig DBA-grensesnitt  
**Verdi:** Forenklede operasjoner

### Funksjonalitet
Menyvalg:
1. Sikkerhetskopier database
2. Gjenopprett database
3. Sjekk helse
4. Vis tablespace-bruk
5. Kjør REORG
6. Samle statistikk
7. Vis logger
8. Administrer brukere

**Målgruppe:** Mindre erfarne DBAer

### Tidsbesparelse
- **Produktivitet:** Ingen behov for å huske kommandoer
- **Opplæring:** Raskere DBA-onboarding
- **Tidsbesparelse per kjøring:** ~180 minutter (manuelt arbeid eliminert)
