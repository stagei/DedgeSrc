# Db2-Restore

**Kategori:** DatabaseTools | **Tidsbesparelse:** ~85 min/kjøring

---

## 🎯 Forretningsverdi

**Problem:** Manuell databasegjenoppretting = risikabelt, tidkrevende  
**Løsning:** Automatisert Db2-databasegjenoppretting  
**Verdi:** Rask, pålitelig katastrofegjenoppretting

### Funksjonalitet
Gjenopprettingsprosess:
1. Validerer backupfil
2. Sjekker diskplass
3. Gjenoppretter database
4. Ruller frem logger (hvis nødvendig)
5. Verifiserer integritet
6. Samler statistikk
7. Sender varsel

**Opsjoner:** Tidspunkt-for-gjenoppretting

### Tidsbesparelse
- **RTO:** 4-6 timer → 45 min (Recovery Time Objective)
- **Pålitelighet:** Null gjenopprettingsfeil
- **Kritisk:** DR-kapabilitet
- **Tidsbesparelse per kjøring:** ~85 minutter (manuelt arbeid eliminert)
