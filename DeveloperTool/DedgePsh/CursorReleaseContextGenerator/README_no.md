# CursorReleaseContextGenerator

**Kategori:** GitTools | **Tidsbesparelse:** ~349 min/kjøring

---

## 🎯 Forretningsverdi

**Funksjon:** Generer release-kontekst for Cursor AI  
**Formål:** AI-assistert release notes  
**Verdi:** Automatisert release-dokumentasjon

### Funksjonalitet
Genererer kontekst:
- Commits siden siste release
- Endrede filer
- Forfattere
- Issue-linker
- Kode diff-sammendrag

**Output:** Markdown for Cursor AI til å generere release notes

### Tidsbesparelse
- **Tid:** 2 timer manuelt → 10 min automatisert
- **Kvalitet:** Komplett endringssporing
- **Releaser:** ~25/år
- **Tidsbesparelse per kjøring:** ~349 minutter (manuelt arbeid eliminert)
